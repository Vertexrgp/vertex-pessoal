Quero corrigir o módulo Financeiro, criando uma base estruturada de Categorias e Contas, pois hoje os campos de seleção estão vazios.

OBJETIVO
Permitir que o usuário cadastre categorias e contas e utilize essas informações nos lançamentos financeiros.

====================================================
1) CATEGORIAS
====================================================

Criar um sistema de categorias financeiras.

Cada categoria deve ter:

- nome
- tipo:
  - despesa
  - receita
- cor (para visual)
- ícone (opcional)

Exemplos padrão:
- Alimentação
- Moradia
- Transporte
- Lazer
- Saúde
- Investimentos
- Salário

Permitir:

- criar
- editar
- excluir

====================================================
2) CONTAS
====================================================

Criar sistema de contas financeiras.

Cada conta deve ter:

- nome
- tipo:
  - conta corrente
  - cartão de crédito
  - carteira
- saldo inicial
- banco (opcional)

Exemplos:
- Nubank
- Itaú
- Dinheiro
- Inter

Permitir:

- criar
- editar
- excluir

====================================================
3) INTEGRAÇÃO COM LANÇAMENTOS

- dropdown de conta deve puxar do banco

- mostrar botão rápido: "Criar conta"
