====================================================

- banco/emissor

- bandeira

Exemplo:
- Banco: Itaú
- Cartão: Platinum
- Bandeira: Visa

2. No cadastro de cartões, incluir campos:
- nomeCartao
- banco
- bandeira
- limiteTotal
- diaFechamento
- diaVencimento
- corPrimaria
- corSecundaria
- ativo

3. Bandeiras disponíveis:
- Visa
- Mastercard
- Elo
- American Express
- Hipercard
- Diners
- Outra

4. Criar identidade visual melhor para os cards dos cartões:
- visual premium
- bordas arredondadas
- boa hierarquia
- nome do cartão em destaque
- banco menor
- bandeira em pill/tag visual
- usar cor coerente com o cartão cadastrado

5. Melhorar os cards atuais:
- não deixar todos com aparência muito genérica
- usar a cor do cartão para o fundo ou gradiente
- exibir bandeira com mais clareza
- exibir nome do cartão e banco com mais elegância

====================================================
2) LISTA DE LANÇAMENTOS — COLUNA PRÓPRIA DE PARCELA
====================================================

Na tela de Lançamentos:

1. Criar uma coluna própria chamada:
Parcela

2. Regras da coluna:
- se compra parcelada: mostrar "1/12", "2/12", etc
- se compra no crédito à vista: mostrar "À vista"
- se não se aplica: mostrar "-"

3. Remover a dependência de mostrar parcela apenas como badge na descrição.

4. A tabela de lançamentos deve ficar assim:
- Data
- Descrição
- Categoria
- Pagamento
- Parcela
- Valor
- Status
- Ações

5. Visual:
- a coluna Parcela deve ficar limpa e elegante
- usar badge/pill discreto
- manter boa legibilidade

====================================================
3) MELHORAR O CADASTRO DE LANÇAMENTOS PARA CARTÃO
====================================================

No modal de Novo Lançamento:

Se formaPagamento = Crédito:
mostrar:
- cartão
- tipoCredito (À vista / Parcelado)

Se Parcelado:
mostrar:
- número de parcelas
- data da primeira parcela

Garantir que:
- o lançamento gere parcelas corretamente
- a coluna Parcela na listagem funcione
- a Fatura do Cartão entenda essas parcelas
